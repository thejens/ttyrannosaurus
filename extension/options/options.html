ttyrannosaurus

Type ! in the address bar to open a terminal.

Sidebar

Controls when the sidebar is shown. The toolbar icon is always a manual override.

tmux

When enabled, each session runs inside a tmux session — terminals survive daemon restarts and can be reconnected from any tab.

Session Names

Uses the on-device Gemini Nano model (Chrome 138+) to generate short labels from terminal output.

Terminal Theme

Paste your Ghostty config (or just the relevant lines — font, colors, palette). Applied immediately to new terminal tabs; no daemon restart needed.
